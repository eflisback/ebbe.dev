# ebbe.dev Portfolio Website

A portfolio website I created to showcase personal projects, skills, and experience. This website is built using React, written in Typescript, and powered by Vite.

The website is live and can be accessed at [https://ebbe.dev](https://ebbe.dev).

## Technologies

- React
- Typescript
- Vite

## Libraries

Some of the libraries that were used in developing the website:

- [React Icons](https://react-icons.github.io/react-icons/)
- [React Router](https://reactrouter.com/)
- [OpenAI API](https://platform.openai.com/docs/)
